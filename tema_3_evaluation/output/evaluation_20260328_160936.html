What is App-ID in Palo Alto Networks and how does it work?

Expected:

App-ID is a patent-pending traffic classification technology in PAN-OS. It identifies applications regardless of port, protocol, or encryption. App-ID uses four classification mechanisms: application signatures, application protocol decoding, heuristics, and SSL/SSH decryption. It continuously reassesses traffic as the session evolves. App-ID enables policy enforcement based on the actual application, not just port numbers, significantly improving security posture.
